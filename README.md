# Roblox Fly Command

Sistema simples de comando `/fly` para Roblox.

## Comandos

- `/fly` → ativa voo
- `/unfly` → desativa voo

## Recursos
- Sistema leve
- Fácil de editar
- Compatível com admin systems

## Instalação

1. Copie o arquivo `FlyCommand.lua`
2. Execute no seu sistema admin/executor
3. Use os comandos no chat
